# Інтеграли — від нуля до інтегрування частинами

> Коротко і по простому. Кожен щабель — одна ідея.
> Запис формул — звичайними символами (без розмітки), як у зошиті.

---

## Драбинка понять

### 1. Функція
Правило «вкинув число → отримав число».
Напр. y = x² : вкинув 3 → отримав 9.

### 2. Графік
Та сама функція, намальована лінією. По горизонталі x, по вертикалі y.

### 3. Швидкість зміни (нахил)
Коли x трохи росте — наскільки росте y? Це «крутизна» графіка в точці.
- На прямій: нахил = (зміна y) / (зміна x).
- На кривій: нахил різний у кожній точці.

### 4. Похідна
Той самий нахил, але точний, у конкретній точці.
Позначення: y′ або dy/dx (відношення нескінченно малих).

Базові похідні, які треба знати:

    (x²)′ = 2x
    (xⁿ)′ = n·x^(n−1)
    (sin x)′ = cos x
    (cos x)′ = −sin x
    (eˣ)′ = eˣ
    (ln x)′ = 1/x

### 5. Диференціал
Переписана похідна: dy = y′·dx.
«Маленький приріст функції = похідна × маленький приріст x».

### 6. Антипохідна (первісна)
Зворотна задача: дано похідну — знайди саму функцію.
Напр. похідна = 2x → функція = x² (+ константа C, бо похідна константи = 0).

### 7. Інтеграл
Знак для антипохідної:

    ∫ 2x dx = x² + C

Знак ∫ — це витягнута S, «сума». Інтеграл склеює всі нескінченно малі
шматочки dy назад у функцію:  ∫ dy = y.

### 8. Геометричний зміст
Інтеграл = площа під графіком функції.
(Сума нескінченно тонких стовпчиків.)

### 9. Базові правила інтегрування (розвернуті похідні з п.4)

    ∫ xⁿ dx     = x^(n+1)/(n+1) + C
    ∫ cos x dx  = sin x + C
    ∫ sin x dx  = −cos x + C
    ∫ eˣ dx     = eˣ + C
    ∫ (1/x) dx  = ln x + C

### 10. Коли просте правило не працює — методи
Якщо під інтегралом **добуток** двох різних функцій (напр. x·cos x) —
звичайне правило не бере. Тут вмикається **інтегрування частинами**.

---

## Інтегрування частинами

### Формула

    ∫ u dv = u·v − ∫ v du

### Звідки береться
Це правило похідної добутку, прочитане навпаки.

    d(u·v) = u dv + v du      ← приріст добутку
    ∫ u dv = u·v − ∫ v du     ← проінтегрували й перенесли

### Навіщо
Щоб складний інтеграл замінити простішим.
За u беремо те, що при похідній **спрощується**.

### Правило вибору u (мнемоніка LIATE)
Що вище в списку — те береш за u:
- **L** — логарифмічні (ln x)
- **I** — обернені тригонометричні (arcsin, arctg)
- **A** — алгебраїчні (x, x², многочлени)
- **T** — тригонометричні (sin, cos)
- **E** — експоненційні (eˣ)

### Приклад: ∫ x·cos x dx

    u = x          →   du = dx
    dv = cos x dx  →   v = sin x

    ∫ x·cos x dx = x·sin x − ∫ sin x dx
                 = x·sin x + cos x + C

Чому працює: x при похідній перетворюється на 1 і зникає,
тому інтеграл праворуч стає тривіальним.

## Уся логіка одним рядком
похідна = нахил / швидкість зміни → інтеграл = зворотна операція =
склеїти назад = площа під графіком → частинами = трюк для добутків.

## Символи (щоб впізнавати запис)
- ∫ — інтеграл
- d, du, dv, dx — диференціали («нескінченно малі»)
- · або × — множення
- ⁿ або ^ — степінь (x² або x^2)
- √ — корінь
- π — пі
- ≈ ≤ ≥ ≠ — приблизно / нерівності
- Σ — сума
