# Тренажер — Сходинка 2: корені та від'ємні степені

> Та сама формула, що й на сходинці 1:
>
>     ∫ xⁿ dx = x^(n+1)/(n+1) + C
>
> Фокус лише в тому, що СПОЧАТКУ треба переписати вираз у вигляді степеня x,
> а вже потім застосувати формулу. Корені та дроби — це теж степені.

---

## Шпаргалка: як переписати (вивчити перед розв'язанням)

    √x      = x^(1/2)
    ³√x     = x^(1/3)
    1/x     = x^(−1)
    1/x²    = x^(−2)
    1/√x    = x^(−1/2)
    x·√x    = x^(3/2)

Правило для дробових/від'ємних так само: степінь +1, ділимо на новий степінь.
Ділення на дріб = множення на перевернутий дріб.
Напр.:  x^(1/2) → +1 = x^(3/2),  ділимо на 3/2 = множимо на 2/3 → (2/3)x^(3/2).

⚠️ Виняток: ∫ x^(−1) dx = ∫ (1/x) dx = ln|x| + C   (формула НЕ працює, бо вийшло б ділення на 0).

---

## Завдання (спробуй сама, відповіді — в самому низу)

1)  ∫ √x dx

2)  ∫ 1/x² dx

3)  ∫ (x² + 1/√x) dx

4)  ∫ (4x³ − 2/x³) dx

5)  ∫ ³√x dx

6)  ∫ (6√x + 5) dx

7)  ∫ 1/x dx          ← пастка! згадай виняток зі шпаргалки

8)  ∫ (x·√x) dx

---

## Алгоритм на кожен приклад
1. Переписати корінь/дріб як степінь x^(...).
2. До степеня додати 1.
3. Поділити коефіцієнт на новий степінь.
4. Не забути + C.
5. Перевірка: взяти похідну від відповіді → має вийти те, що було під інтегралом.

## ВІДПОВІДІ (підглядати після спроби!)

1)  ∫ √x dx = ∫ x^(1/2) dx = x^(3/2)/(3/2) + C = (2/3)·x^(3/2) + C
       = (2/3)·x·√x + C

2)  ∫ 1/x² dx = ∫ x^(−2) dx = x^(−1)/(−1) + C = −1/x + C

3)  ∫ (x² + 1/√x) dx = x³/3 + ∫ x^(−1/2) dx
       = x³/3 + x^(1/2)/(1/2) + C = x³/3 + 2√x + C

4)  ∫ (4x³ − 2/x³) dx = 4·x⁴/4 − 2·∫ x^(−3) dx
       = x⁴ − 2·x^(−2)/(−2) + C = x⁴ + 1/x² + C

5)  ∫ ³√x dx = ∫ x^(1/3) dx = x^(4/3)/(4/3) + C = (3/4)·x^(4/3) + C

6)  ∫ (6√x + 5) dx = 6·x^(3/2)/(3/2) + 5x + C = 4·x^(3/2) + 5x + C
       = 4·x·√x + 5x + C

7)  ∫ 1/x dx = ln|x| + C        (той самий виняток!)

8)  ∫ x·√x dx = ∫ x^(3/2) dx = x^(5/2)/(5/2) + C = (2/5)·x^(5/2) + C
